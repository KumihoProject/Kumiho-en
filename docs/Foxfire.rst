.. _Foxfire:
Foxfire
====

``Foxfire`` is a browser extension needed to use Kumiho platform. It provides a proxy for ``*.klay`` domains and injects :ref:`Ahri<Ahri>` SDK into your
browser.

- Chrome_
- Firefox_

.. _Chrome: https://chrome.google.com/webstore/detail/foxfire/pccgbfogagfobhemhlibegioklnkihgk
.. _Firefox: https://addons.mozilla.org/ko/firefox/addon/kumiho/
